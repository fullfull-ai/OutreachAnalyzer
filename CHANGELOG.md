# Changelog

## 1.0.0 — 2026-02-28

Initial release.

- LinkedIn data export analysis (Connections.csv + messages.csv)
- Outreach detection — identifies conversations you initiated
- Response matching with timing metrics
- Multi-message outreach merging
- Template clustering via fuzzy matching (rapidfuzz)
- Greeting stripping (English + Hebrew) and name normalization
- Persona classification — automatic fuzzy clustering or AI-powered via OpenAI API
- Per-template and per-persona response rate breakdowns
- Insights: response rate by day of week, message length, and response speed
- HTML report generation
- Excel export of raw results
- Configurable thresholds via config.yaml
